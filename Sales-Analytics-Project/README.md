# Sales Analytics Project
This project performs deep analysis of sales performance using Python, SQL, and data visualization.  
It includes a Streamlit dashboard and a demo webpage to showcase results.

## Features
- Data cleaning & preprocessing
- Exploratory Data Analysis (EDA)
- SQL-driven insights
- Automated KPIs and charts
- Streamlit dashboard to visualize sales trends
- Static webpage to present dashboard screenshots and insights
- Organized folder structure for resume-ready presentation

## Tech Stack
- Python (pandas, numpy, matplotlib, seaborn, plotly)
- SQL (MySQL/PostgreSQL)
- Jupyter Notebook
- Streamlit
- HTML/CSS/JS (for dashboard showcase webpage)
- GitHub

## Project Workflow
1. Place raw data inside `/data`
2. Perform EDA inside `/notebooks/EDA.ipynb`
3. Write analytical SQL queries inside `/sql/queries.sql`
4. Build interactive dashboard using `/dashboard/app.py`
5. Showcase KPIs & images using `/web/index.html`

## Dashboard Preview
- Regional sales distribution  
- Product performance  
- Revenue time series  
- Customer segmentation  
- Top-selling categories  

## How to Run
### Install dependencies:
```bash
pip install -r requirements.txt
```

### Run Dashboard:
```bash
streamlit run dashboard/app.py
```
